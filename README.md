## 关于此项目

此项目是基于原项目 [zhheo/HeoWeb](https://github.com/zhheo/HeoWeb) 进行修改，并移植至Hexo

## 安装

 ```BASH
git clone -b master https://github.com/rootlexblog/Heoweb.git themes/heoweb
 ```
